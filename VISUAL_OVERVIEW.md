## 📊 Data Flow Example: Submit Typing Session

```
User types text in Practice Page
        ↓
React Component calculates WPM, accuracy, errors
        ↓
Calls typingService.submitSession()
        ↓
Axios sends POST /api/typing/submit with JWT token
        ↓
Backend: protect middleware validates JWT
        ↓
Backend: typing.controller.submitSession()
        ↓
Creates Progress record in MongoDB
        ↓
Updates User typingStats
        ↓
Returns success response with updated stats
        ↓
Frontend: React Query updates cache
        ↓
UI displays success toast and updated statistics
        ↓
Dashboard statistics automatically refresh
```

## 🔐 Authentication Flow

```
1. User Registration
   ├─ POST /api/auth/register
   ├─ Hash password with bcrypt
   ├─ Create user in database
   ├─ Generate JWT token
   └─ Return user + token

2. User Login
   ├─ POST /api/auth/login
   ├─ Verify email and password
   ├─ Generate JWT token
   ├─ Store token in localStorage
   └─ Redirect to dashboard

3. Protected Requests
   ├─ Include token in Authorization header
   ├─ Backend validates JWT
   ├─ Attach user to request object
   └─ Allow/deny access based on role
```
